1. APP OVERVIEW
================================================================================

Product (UI name):     StartupForge
Repo/README name:      Agent Autopilot  <-- naming split; confirm before logo work
Tagline:               "Describe your idea. The agent does the rest."
Secondary tagline:     "Describe your idea. Review each step before proceeding."
Loading copy:          "StartupForge is thinking"

What the app does:
  - User describes a product idea in plain text
  - AI agent runs 4 steps: Competitor Research, Landing Page, Launch Posts,
    AI Agent Prompts
  - User reviews each step before proceeding
  - Outputs: markdown research, HTML landing page, social post drafts, agent prompts

Tech stack:            Next.js 14, React, Tailwind CSS, Clerk auth, Supabase
Theme:                 Forced dark mode (html.dark)
Current asset state:   ZERO image files in repo; no public/ folder; no favicon/OG
